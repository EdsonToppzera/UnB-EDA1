# UnB - Estrutura de Dados 1 (EDA 1)

Este repositório contém as atividades práticas, exercícios de laboratório e projetos desenvolvidos durante a disciplina de **Estrutura de Dados 1 (EDA 1)** do curso de Engenharia de Software da Universidade de Brasília (UnB - Campus Gama).

O objetivo principal é consolidar os fundamentos de manipulação de memória, ponteiros e estruturas de dados dinâmicas na linguagem C.

## 🚀 Tecnologias Utilizadas

* **Linguagem C** (Padrão ANSI / C11)
* **Compilador:** GCC (MinGW-w64 via MSYS2)
* **Ambiente:** Visual Studio Code
* **Controle de Versão:** Git e GitHub

## 📚 Conteúdos Desenvolvidos

Com base no plano de ensino da disciplina, o repositório está organizado para cobrir os seguintes tópicos:

* **Funções, Iteração e Recursividade:** Resolução de problemas via algoritmos iterativos e recursivos.
* **Vetores, Matrizes e Structs:** Agrupamento e organização de dados estruturados heterogêneos.
* **Ponteiros e Manipulação de Endereços:** Compreensão profunda sobre referências de memória.
* **Manipulação de Arquivos:** Leitura e escrita de dados persistentes (`.txt` e binários).
* **Alocação Dinâmica de Memória:** Gerenciamento manual da memória Heap (`malloc`, `calloc`, `realloc` e `free`).
* **Complexidade Computacional:** Análise de desempenho de algoritmos utilizando a Notação Big-O.
